Técnicas

7 Camadas de Animação

01 // Stagger Reveal

Grid cards entram em cascata com variants + staggerChildren

02 // Letter-by-Letter

h2 fade-up por letra com whileInView + split spans

03 // CountUp

Números animam do 0 ao valor final com react-countup

04 // Magnetic Buttons

Botões seguem o rato com useMotionValue

05 // Parallax Scroll

h2 responde ao scroll com useScroll + useTransform

06 // SVG Timeline

Linha progressiva entre steps com stroke-dashoffset

07 // Hover Cards

Cards reagem com whileHover scale + rotate + shadow
